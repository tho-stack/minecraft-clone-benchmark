Minecraft Voxel Game - Documentation

Architecture Overview

This is a complete Minecraft-style voxel sandbox game built as a single self-contained HTML file using Three.js for 3D rendering. The game implements a chunk-based world with procedural terrain generation.

Core Systems

  • Chunk System: World divided into 16x64x16 voxel chunks, independently meshed as Groups
  • Procedural Generation: Seeded Simplex noise for terrain, caves, biomes
  • Rendering: Custom mesh generation with face culling, vertex colors, procedural water
  • Physics: AABB collision detection, gravity, jumping, flying mode
  • Input: Pointer Lock API for mouse capture, WASD movement

Technical Decisions

Chunk-Based Architecture

World divided into 16x64x16 voxel chunks. Each chunk is independently generated, meshed as a Three.js Group (solid + water), and loaded/unloaded based on player position. This enables infinite terrain via streaming.

Simplex Noise Terrain

Layered 2D noise at different frequencies creates natural rolling hills. 3D noise with threshold carving creates cave systems below y=45. Biome variation uses lower-frequency noise: forest biome has grass surface and trees; plains biome has sand surface.

Face Culling Strategy

Only faces between solid blocks and air/water are rendered. This reduces triangle count from O(blocks) to O(surface area), enabling smooth performance at render distance 8.

Transparent Water Rendering

Water uses a separate mesh in the chunk Group with transparent material (opacity 0.7). Only top water surface is rendered when water is exposed to air, optimizing fill rate.

Seeded Determinism

World uses seed 1337 via Simplex noise. The spawn-area hash (displayed in HUD) confirms byte-for-byte identical terrain on every reload.

Procedural Textures

All block textures generated at runtime via Canvas 2D. Each block type has distinct patterns with per-face variation (grass top differs from sides, wood top differs from sides). Textures are wired into Three.js via MeshLambertMaterial map property. Deterministic via per-texture seeded RNG (seed 1337). No external texture files required.

How to Play

  1. Open game.html directly in Chrome, Edge, or Firefox (no server needed)
  2. Click the canvas to capture mouse
  3. Use WASD to move, mouse to look around
  4. Left click breaks blocks, right click places
  5. Press F3 to toggle debug HUD (visible by default)
  6. Press E for inventory/palette
  7. Press F to toggle fly mode
  8. Press 1-9 or scroll to select block type
  9. ESC releases mouse and shows pause overlay
